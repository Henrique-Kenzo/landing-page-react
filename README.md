# 🚀 Landing Page - Aprendizado Full-Stack

Este projeto nasceu de uma vontade real de parar de apenas assistir tutoriais e começar a **aprender fazendo**. O objetivo principal foi consolidar conceitos de desenvolvimento Full-Stack utilizando as tecnologias mais modernas do ecossistema JavaScript/TypeScript.

## 🧠 A Jornada de Aprendizado

Diferente de seguir um passo a passo pronto, este projeto foi construído através de um processo de **brainstorming ativo**. Eu queria entender não só _como_ rodar um comando, mas _por que_ estamos usando Vite, como o Tailwind v4 simplifica o CSS e como o React se integra de verdade com uma API em Node.js.

### O que eu consolidei aqui:

- **Frontend Moderno:** Migração para o **Tailwind CSS v4**, entendendo a nova configuração baseada no Vite.
- **Tipagem Real:** Uso de TypeScript tanto no cliente quanto no servidor para evitar erros bobos de comunicação.
- **Fluxo de Dados:** Criação de um formulário controlado com `useState` e integração com o backend via `fetch` assíncrono.
- **Mentalidade Full-Stack:** Entender que o Frontend e o Backend são duas aplicações separadas que "conversam" através de contratos (JSON).

## 🛠️ Tecnologias Utilizadas

### Frontend

- **React + TypeScript** (via Vite)
- **Tailwind CSS v4** (estilização rápida e responsiva)
- **Lucide React / Heroicons** (para ícones)

### Backend

- **Node.js + Express**
- **TSX** (para rodar TypeScript sem complicação no ambiente de dev)
- **CORS** (para permitir a comunicação entre as portas 5173 e 3000)
